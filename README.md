# Objective-C Primer for Swift Developers

## Printing to the Console 

**Objective-C** 
```objective-c
NSLog(@"Hello, World");
```

**Swift** 
```swift 
print("Hello, World") 
```

## NSString

```objective-c
NSString *message = @"Welcome to Objective-C";
NSLog(@"%@", message);

// Welcome to Objective-C
```
